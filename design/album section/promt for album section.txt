Design ONLY the "Memory Gallery" section for a premium digital graduation memory website.

Do NOT design any other section.

This is the largest section of the website.

The gallery contains memories collected over four university years.

Unlike previous sections, this section is image-focused.

Visitors should feel like browsing through a beautifully curated digital photo album.

==================================================
STYLE
==================================================

Follow the established Design System.

Visual references:

• Apple Photos
• Pinterest Editorial
• VSCO Gallery
• Premium Photography Portfolio
• Korean Graduation Photobook

Avoid:

• Instagram clone
• Facebook feed
• Social media appearance
• Dashboard
• Masonry with uneven gaps
• Bright colorful UI

The photographs should always remain the visual focus.

==================================================
BACKGROUND
==================================================

Warm White (#FFFDF9)

No decorative illustrations.

No doodles.

No scrapbook elements.

The gallery should feel clean.

Allow the photographs to speak.

==================================================
SECTION HEADER
==================================================

Centered.

Large title:

Memory Gallery

Subtitle:

Every ordinary moment
became an unforgettable memory.

Large whitespace.

==================================================
LAYOUT
==================================================

Create a clean masonry gallery.

Different image sizes.

Different aspect ratios.

Perfect spacing.

16–24px gaps.

Images should naturally create rhythm.

Do NOT crop photos excessively.

Keep original composition whenever possible.

Maximum content width:

1400px.

==================================================
PHOTO CARD
==================================================

Each photo has:

Rounded corners (20px)

Soft shadow

White background

No border

No visible caption by default.

==================================================
HOVER
==================================================

Hovering a photo:

Scale to 1.02.

Shadow increases slightly.

Image brightness increases by 5%.

A small icon appears
at the bottom-right:

Expand

The interaction should feel elegant.

==================================================
CLICK
==================================================

Clicking a photo opens
a premium fullscreen modal.

Background fades to soft black
with slight blur.

The modal should resemble
Apple Photos rather than Instagram.

==================================================
PHOTO MODAL
==================================================

Maximum width:

1400px.

Split into two columns.

LEFT (70%)

Large photo.

Maintain original aspect ratio.

Do not crop.

Allow zoom.

RIGHT (30%)

Scrollable information panel.

White background.

Large padding.

Rounded corners.

==================================================
PHOTO INFORMATION
==================================================

Top:

Photo title

Small description

Upload date

Uploader name

Typography remains minimal.

==================================================

==================================================

Below the description,

display comments.

Every comment displays:

Avatar

Name

Comment

Timestamp

Comments appear inside soft rounded paper cards.

Spacing between comments:

24px.

Do NOT make comments look like
Instagram or Facebook.

Instead,

they should resemble handwritten memory notes.
